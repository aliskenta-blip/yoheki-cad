yoheki-cad v0.3 使い方

基本操作:

  • ツールバーで描画モードを選択
  • キャンバスをクリック&ドラッグで描画
  • 選択モード(S)でオブジェクトをクリックして編集

線・図形:

  • L 直線：始点→終点をクリック
  • R 矩形：ドラッグで描画
  • C 円：ドラッグで半径指定
  • 多角形：クリック連続→Enterまたは右クリックで完了
  • フリーハンド：マウス押しっぱなしで線

寸法線:

  • D 寸法線：始点→終点でドラッグ。長さは自動計算&表示

鉄筋:

  • B 鉄筋：始点→終点でドラッグ。径と端部曲げは左パネルで事前設定
  • 鉄筋を選択 → 右パネルで径・規格・端部曲げを変更可能

🆕 v0.3 スナップ:

  • 左パネル「オブジェクトスナップ」ON時、描画中にカーソルが近くの端点(□)・中点(△)・交点(✕)・垂線(◇)に吸着
  • 緑のマーカーが表示された位置でクリックすると正確にスナップ
  • オブジェクトスナップが優先、無ければグリッドスナップ

🆕 v0.3 配筋オートツール (🔩メニュー):

  • U字キャップ生成：主筋2本を選択 → 頭部をU字で連結(W4キャップ筋相当)
  • ボックス形成：上下2本を選択 → 両端90°曲げで閉鎖ボックス(底版B1/B2相当)
  • ピッチ配置：1本を選択 → ピッチ・本数・方向を指定して等間隔複製

🆕 v0.3 配筋検査 (🔍メニュー):

  • 全鉄筋のかぶり厚(構造体外形からの距離)と定着長(径×係数)を一覧チェック
  • 不足する鉄筋は赤丸でハイライト表示
  • 最小かぶり・定着長係数はモーダル上部で変更可能

編集:

  • 選択 → ドラッグで移動
  • 選択 → 四隅でリサイズ、上端で回転
  • Delete 削除
  • Ctrl+D 複製
  • Ctrl+Z / Ctrl+Y Undo/Redo
  • Ctrl+A 全選択

表示:

  • マウスホイール：ズームイン/アウト
  • Space+ドラッグ：パン
  • F：全体表示

保存:

  • 💾 JSON保存：編集状態を完全に保存
  • SVG/PNG出力：画像として出力
  • テンプレート：v0.1のパラメトリック擁壁を読み込み

テンプレート読み込み

パラメトリック設計したテンプレートを編集可能なオブジェクトとしてキャンバスに展開します。

🔩 配筋オートツール

鉄筋を選択してから各機能を実行します。複数選択は Shift+クリック。

① U字キャップ生成

主筋2本を選択 → 頭部をU字(⊓)で連結。壁頭部のキャップ筋(W4相当)を自動生成します。

② ボックス形成 (⊓+⊔)

上端筋・下端筋となる2本を選択 → 両端を90°曲げて閉鎖ボックスを形成(底版B1/B2相当)。

③ ピッチ配置 (連続複製)

鉄筋1本を選択 → 指定方向・ピッチ・本数で等間隔に複製します。@マーク付きの配力筋・主筋並びに。
