# yoheki-cad（擁壁CAD）

待ち受け式擁壁などの構造図を作図するブラウザアプリ。単一HTMLファイルで動作し、ビルド・依存パッケージは不要です。`index.html` をブラウザで開いて使用します。

## バージョン履歴

| タグ | 内容 |
|---|---|
| v0.6.1 | 印刷・PNG出力の修正 |
| v0.6 | ツールチップ・ガイド・3Dパース |
| v0.3 | スナップ・配筋ツール・検査 |
| v0.2 | フリーハンド編集対応 |
| v0.1 | パラメトリック擁壁エディタ（試作版） |

各バージョンは git タグ（`v0.1`〜`v0.6.1`）で参照・チェックアウトできます。現在の `index.html` は最新版（v0.6.1）です。

## 使い方

`index.html` をダブルクリックしてブラウザ（Chrome/Edge 等）で開きます。

## 開発メモ

- 元ファイルは `Downloads` に `yoheki-cad_vX.Y.html` として保存されていたものを、このリポジトリで一元管理する形に移行しました。
- 今後の変更は `index.html` を編集し、区切りのよいところで git タグを付けてバージョンを管理します。
